# damo L45 Keypoint Head CNN Deployment
Andrew Price 23.03.2026

This repo contains code to run inference for the damo L45 Keypoint CNN as part of the CVLab - Klepsydra fault tolerance project.
More work can be done to compress the damo yolo backbone repo, as only a single architecture is used.
